# TripMaster — лендинг сервиса планирования путешествий

**Тема проекта:** Лендинг для сервиса планирования путешествий (поиск отелей, билетов и маршрутов).

**Использованный CSS-фреймворк:** Bulma

## Описание
Одностраничный сайт (лендинг), представляющий концепцию сервиса TripMaster. Содержит:
- навигационную панель;
- герой-секцию с призывом к действию;
- блок с описанием проекта;
- преимущества в виде карточек;
- отзывы пользователей;
- футер с контактами и ссылками на соцсети.
